#

## zsh快捷键

<b type="h">- 移动</b>
Ctrl + a：移动到行首
Ctrl + e：移动到行尾
Ctrl + f：向前移动
Ctrl + b：向后移动

<b type="h">- 查找命令</b>
Ctrl + p：上一条命令
Ctrl + n：下一条命令
Ctrl + r：搜索历史命令
Ctrl + y：召回最近用命令删除的文字

<b type="h">- 删除</b>
Ctrl + u：清空当前行
Ctrl + h：删除光标之前的字符
Ctrl + d：删除光标所指的字符
Ctrl + w：删除光标之前的单词
Ctrl + k：删除从光标到行尾的内容

<b type="h">- 替换</b>
Ctrl + t：交换光标和之前的字符
