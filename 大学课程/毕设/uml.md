## 用例图

```puml
@startuml
left to right direction
skinparam packageStyle rectangle

actor 教师
actor 学生
actor 管理员

rectangle 作业管理系统 {
    教师 --> (使用账号)
    学生 --> (使用账号)
    (使用账号) ..> (登录账号)       : include
    (使用账号) ..> (注册账号)       : include
    (登录账号) <.. (修改账号信息)   : <<extend>>

    教师 --> (使用作业)

    (使用作业) ..> (发布作业)           : include
    (发布作业) <.. (定义、使用作业模板)  : <<extend>>
    (使用作业) ..> (作业的增删改查)      : include
    (使用作业) ..> (批改作业)           : include
    (使用作业) ..> (统计作业完成情况)    : include

    教师 --> (创建、管理课程)
    教师 --> (创建、管理班级)


    学生 --> (完成作业)
    (完成作业) ..> (编辑作业)       : include
    (编辑作业) <|-- (在线编辑)
    (编辑作业) <|-- (文档上传)
    (完成作业) ..> (管理个人作业)   : include
    (完成作业) ..> (统计个人所有作业): include

    教师 --> (消息提醒)
    学生 --> (消息提醒)
    (消息提醒) ..> (开启/关闭消息提醒)  : include
    (消息提醒) ..> (定义提醒的内容)     : include
    (消息提醒) ..> (定义提醒的触发条件)  : include

    管理员 --> (账号、角色管理)
    管理员 --> (课程、班级管理)
    管理员 --> (作业信息管理)

}
@enduml

```

## 时序图

### 登录

```puml
@startuml
autonumber

actor user
user -> 前端登录界面 :进行登录操作
前端登录界面 -> admin系统 :进行登录校验 
前端登录界面 <- admin系统 :返回登录结果
user <- 前端登录界面 :显示登录结果

@enduml
```

### 退出

```puml
@startuml
autonumber

actor user
user -> 前端界面 :进行退出操作
user <- 前端界面 :弹窗“确认”窗口
user -> 前端界面 :确认操作
前端界面 -> admin系统 :清除登录状态
前端界面 <- admin系统 :返回结果
user <- 前端界面 :退出账号

@enduml
```
