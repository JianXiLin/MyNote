## Java10

### 1. 引入var类型

通过引入var来实现局部变量的类型判断。即局部变量的声明不需要指定参数类型，将由实例化的类型来自动推断。

```java
var s = "test"; //String
var list = new ArrayList<String>();
```

<b type="h">注意点：</b>

以下情况无法使用var

1. 使用var时，无赋值。
2. 使用var替换左边的函数式接口
3. 作为方法的返回类型或参数、构造器参数类型
4. 用于属性声明
    原因：属性产生默认值需指定类型
5. 数组的静态初始化

```java
// error
var array = {1,2,3}
```
