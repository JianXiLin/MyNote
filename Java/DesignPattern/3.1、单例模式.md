 ### <font color=#4CAF50 size=4>1、 饿汉式(静态常量) </font>
 #### 步骤：
1. 构造器私有化, 外部能new
2. 2.本类内部创建对象实例
 #### 代码：
```java
/**	 
 *  饿汉式(静态变量)
 */

class Singleton {
	/**
	 * 1. 构造器私有化, 外部能new
	 */
	private Singleton() {
		
	}

	/**
	 * 2.本类内部创建对象实例
	 */
	private final static Singleton instance = new Singleton();

	/**
	 * 3. 提供一个公有的静态方法，返回实例对象
	 * @return
	 */
	public static Singleton getInstance() {
		return instance;
	}
	
}
```
