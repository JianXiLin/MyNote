# 树

## 1. 红黑树

### 1.1. 红黑树的由来

### 1.2. 特点

1️⃣ 各节点为黑色或红色
2️⃣ 根节点为黑色
3️⃣ 叶节点为黑色
4️⃣ 插入点为红色
5️⃣ 红色节点不相连

### 1.3. 变化规则

1️⃣ 改变颜色

红色变为黑色、黑色变为红色

#### 使用条件

当前节点、父节点、叔节点都为红色

2️⃣ 左旋

右子节点变为父节点、父节点变为左子节点。

#### 使用条件

当前节点、父节点为红色。叔节点为黑色
当前节点为父节点的右子树

3️⃣ 右旋

#### 使用条件

当前节点、父节点为红色。叔节点为黑色
当前节点为父节点的左子树
