# 一、 线性布局
#### 1、自身属性：
    Android：gravity 对齐方式             
    Android：ignoreGravity 那些组件不受gravity影响 
    Android：orientation 流式布局方向 v 垂直
#### 2、子属性		
Android：weight 权重 （剩下的部分按权重分）		
dp 长度单位（根据屏幕）
